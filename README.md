MERN Stack E-Commerce Platform
​A fully functional backend-driven e-commerce application built with Node.js, Express, and MongoDB Atlas. This project features user authentication, product management, and dynamic search capabilities.
​🚀 Live Demo
​Vercel Link: https://ecommerce-backend-design-five.vercel.app/
API Endpoint (Seed Data): /seed (To reset and populate database)
​✨ Key Features
​User Authentication: Secure Signup, Login, and Session management using bcryptjs and express-session.
​Dynamic Product Listing: Server-side pagination (5-8 products per page) for optimized loading.
​Real-time Search: Integrated search bar to filter products by name or category.
​Cloud Database: Powered by MongoDB Atlas with seamless connection handling.
​Shopping Cart: Persistent session-based cart functionality to add/manage items.
​Responsive UI: Built with EJS (Embedded JavaScript) for a dynamic and mobile-friendly experience.
